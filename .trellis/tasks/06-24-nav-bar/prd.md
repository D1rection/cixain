# 实现顶部导航栏

## 目标

实现站点顶部导航栏，包含站点名和导航链接。

## 具体内容

1. `src/components/NavBar.jsx` + `NavBar.module.css`
   - 左侧显示站点名 "cixain"
   - 右侧依次：首页、分类下拉、关于
   - 分类从 `useBlogData()` 的 posts 中动态提取
   - 使用 wouter `Link` 导航
   - 固定顶部，不随滚动消失

2. 提取所有文章的 categories 去重列表

3. 在 App.jsx 中引入 NavBar，所有页面共享

## 验收标准

- [ ] 导航栏在首页、文章页、关于页都可见
- [ ] 分类从实际文章数据动态生成
- [ ] 点击链接正确跳转
- [ ] 导航栏样式克制约简

## 不包含

- 左侧分类列表 — 后续 task
- 标签云 — 后续 task
- 响应式（移动端折叠）— 后续优化
