# 全文搜索

## Goal

导航栏增加搜索图标，点击弹出搜索浮层，实时检索文章标题、描述和正文。

## Requirements

- 导航栏右侧添加搜索图标（🔍）
- 点击后弹出居中搜索浮层
- 输入时实时展示匹配结果，Fuse.js 模糊搜索
- 按 Escape 或点击背景关闭浮层
- 构建时生成搜索索引（标题、描述、正文纯文本）

## Acceptance Criteria

- [ ] 任何页面点击搜索图标弹出浮层
- [ ] 输入关键字实时展示标题/描述/正文匹配结果
- [ ] 点击结果跳转到对应文章
- [ ] Escape 关闭浮层
- [ ] 构建后正确生成 search-index.json
