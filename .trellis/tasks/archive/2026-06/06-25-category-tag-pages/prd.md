# 分类/标签独立页面 + SSG

## Goal

分类和标签从 URL 参数升级为独立路由页面，SSG 预渲染，视觉上更精致。

## Requirements

- 新增路由 `/category/:slug` 和 `/tag/:slug`，展示对应过滤后的文章列表
- 页面头部显示分类/标签名称、文章数量、返回全部链接
- SSG 构建时预渲染所有分类和标签页面
- 侧边栏分类/标签链接指向新的独立路径
- 支持分页

## Acceptance Criteria

- [ ] 访问 `/category/Tech` 显示所有技术类文章
- [ ] 访问 `/tag/react` 显示所有含 react 标签的文章
- [ ] 每个页面有美观的头部（名称、文章数、清除筛选）
- [ ] SSG 构建后生成对应 HTML 文件
- [ ] 侧边栏链接已更新为新路径
