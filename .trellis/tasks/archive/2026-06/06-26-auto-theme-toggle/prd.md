# 主题按时间自动切换 + 按钮重构

## Goal

主题默认按时间自动切换（白天亮色、夜晚暗色），手动点击可覆盖，三态循环。同时升级主题切换按钮的图标和动效。

## Design

**三态循环**：自动 → 亮色 → 暗色 → 自动...
- 首次访问无偏好 → 按时间自动选择
- 手动切亮/暗后保存偏好
- 切回自动时清除 localStorage 中的偏好

**按钮图标**：SVG 替代 emoji，三态各有独立图标，点击有旋转过渡

## Requirements

- 6:00-18:00 自动亮色，其余自动暗色
- 点击按钮循环：自动 → 亮 → 暗 → 自动
- localStorage 只保存显式选择的亮/暗
- inline script 支持时间检测防止闪烁

## Acceptance Criteria

- [ ] 白天首次访问为亮色
- [ ] 夜晚首次访问为暗色
- [ ] 手动切亮/暗后刷新保持
- [ ] 切回自动后回归时间驱动
- [ ] 三态图标可区分且有动效
