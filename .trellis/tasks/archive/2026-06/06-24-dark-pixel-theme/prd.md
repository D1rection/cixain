# 深色像素主题

## 目标

全局暗色像素风主题，CSS 变量驱动一键亮暗切换。

## 具体内容

1. 更新 `src/styles/global.css` — 双主题 CSS 变量
   - 暗色：`#0a0a0a` 底 + `#00ff41` 强调 + 白色文字
   - 亮色：白色底 + `#2563eb` 强调 + 深色文字
   - 通过 `[data-theme="dark/light"]` 切换
2. 像素字体 — 引入 Press Start 2P，仅用于品牌 logo
3. 扫描线/网格背景装饰（暗色模式下）
4. 主题切换按钮 — 简单 toggle，切换 `data-theme`
5. 默认跟随系统偏好 `prefers-color-scheme`

## 验收标准

- [ ] 暗色模式：纯黑底 + 亮绿强调色 + 扫描线效果
- [ ] 亮色模式：白底 + 蓝强调色
- [ ] 品牌 logo 用像素字体
- [ ] 主题切换按钮在导航栏可见
- [ ] 刷新后保持选择（localStorage）
- [ ] 首次访问跟随系统偏好

## 不包含

- 每个组件的 theme 适配 — 后续逐个调整
