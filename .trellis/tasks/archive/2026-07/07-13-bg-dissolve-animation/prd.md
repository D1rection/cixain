# PRD: 背景切换像素溶解动画

## 需求
点击 Navbar 背景切换按钮时，新旧背景图之间播放像素溶解过渡动画，提升交互品质。

## 方案
使用 Canvas 实现 8×8 像素块的随机溶解效果：
- 截取当前 CSS 背景图的 URL
- 在页面上覆盖全屏 canvas
- 将旧图和新图分别绘制到隐藏 canvas 读取 ImageData
- 将图像划分为 8×8 的像素块网格
- 随机排列块顺序，逐块从旧图过渡到新图
- 动画完成后更新 `--bg-image` CSS 变量，移除 canvas

## 涉及文件
- `src/components/NavBar.jsx` — 点击处理器中触发 dissolve 动画
- `src/utils/pixelDissolve.js` — 新增，Canvas 像素溶解核心逻辑

## 验收标准
- [ ] 点击背景切换按钮时，旧图以像素块形式溶解到新图
- [ ] 动画完成后背景图正确切换
- [ ] 动画过程不卡顿（8×8 块级别，非逐像素）
- [ ] 动画期间按钮可正常点击
- [ ] 移动端正常工作
