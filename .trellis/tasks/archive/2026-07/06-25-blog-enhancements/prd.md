# 博客功能优化

## Goal

对博客进行一系列功能增强，提升 SEO、阅读体验和内容发现能力。

## Requirements

- 完成以下独立子任务：

| # | 子任务 | 说明 |
|---|--------|------|
| 1 | SEO 元数据 | OG 标签、sitemap.xml、RSS Feed |
| 2 | 阅读进度条 | 文章页顶部显示滚动进度 |
| 3 | 回到顶部 | 长页面右下角悬浮按钮 |
| 4 | 文章目录 TOC | 自动提取标题生成锚点导航 |
| 5 | 字体加载优化 | Press Start 2P 预加载/非阻塞 |
| 6 | 全文搜索 | 前端搜索文章标题和内容 |

## Implementation Order

建议按依赖关系排序实现：
1. 字体优化（独立，无依赖）
2. SEO 元数据（SSG 构建层改动，独立）
3. 阅读进度条（独立）
4. 回到顶部（独立）
5. 文章目录 TOC（依赖 PostContent 的 HTML 结构）
6. 全文搜索（最后一个，最复杂）

## Notes

- 每个子任务独立验收、独立归档
- 复杂子任务需要 design.md + implement.md
