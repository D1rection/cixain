# PRD: Navbar 背景切换按钮

## 需求
在 Navbar 主题按钮旁新增一个按钮，用于切换/开关全屏像素背景图。

## 设计决定
- **位置**：主题按钮右侧，与搜索、GitHub 等操作按钮同一区域
- **图标**：风景/画板 SVG 图标
- **交互**：循环切换：图1 → 图2 → 图3 → 无背景 → 图1...
- **持久化**：用户选择存入 `localStorage`（key: `cixain-bg`），值为图片索引或 `null`（无背景），页面加载时读取
- **每日轮换逻辑**：未手动切换时，按日期取模选图；手动切换后覆盖
- **当前状态指示**：无背景时按钮浅色/低透明度
- 背景通过 CSS 自定义属性 `--bg-image` 控制

## 涉及文件
- `src/components/NavBar.jsx` — 新增按钮
- `src/components/NavBar.module.css` — 按钮样式
- `index.html` — 更新 inline script，读取 localStorage 替代硬编码轮换

## 验收标准
- [ ] Navbar 主题按钮旁出现背景切换按钮
- [ ] 点击切换：图1 → 图2 → 图3 → 无背景 → 图1
- [ ] 刷新后保持上次选择
- [ ] 无背景时按钮淡色，有背景时正常
- [ ] 亮色/暗色主题下按钮风格一致
