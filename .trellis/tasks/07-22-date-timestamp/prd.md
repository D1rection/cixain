# frontmatter 日期支持时间戳

## Goal

frontmatter 的 `date` 字段支持同时包含时间（`2026-07-22 10:30:00`），作为同一天文章的排序依据，替代当前仅靠日期 + slug 二级排序的方案。

## Requirements

- frontmatter `date` 同时兼容日期和日期+时间两种格式：
  - `date: 2026-07-22`（纯日期，向后兼容）
  - `date: 2026-07-22 10:30:00`（带时间，最佳实践）
- 未来日期过滤逻辑能正确处理两种格式
- 页面展示、RSS、归档均正确工作
- 模板文件中标注新格式

## 涉及文件

- `scripts/build-posts.js` — 未来日期判断兼容两种格式
- `Templates/` — 在文章模板中标注新格式
- 其余文件无需改动（`new Date()`、`toISOString()`、`toLocaleDateString()` 自动兼容）

## 验收标准

- [ ] `date: 2026-07-22` 格式的文章仍被正确解析和展示
- [ ] `date: 2026-07-22 10:30:00` 格式的文章按时间排序
- [ ] 未来日期过滤在两种格式下均正确
- [ ] RSS feed、归档页正确显示
